About me

My name is Emil Dahl-Hansen, I'm a 22 year old software developer located in Sweden, originally born in Norway.
I have been developing software as a hobby for about 4 years in different fields, both back-end and front-end.

Before I started learning how to program, I enrolled at a technical institute in Sweden, with the direction of Civil Engineering.
The course I enrolled had programming as one of the base courses, there I learned how to compile and run some simple Java code.
Since I primarily studied math and physics, I tried applying that knowledge in programming and saw a whole new world open up.
This is what led to the change of career goals.

My strongsuits are being very design oriented around animations through math, specifically to create fun and interactive
elements by engineering the math used to animate those elements.
I also find myself effective at writing and integrating different systems that work together, hence why I decided to learn alot about game development.

Now my endgame goal is to find myself engineering data structures and programming interactive systems in a game studio.


My progress

I started off in the Server and Database fields, where I developed an at-home dedicated server in Minecraft.
This involved setting up a database for storing player data to track their progress
as well as handling server side events for gameplay interactions.
(For reference, go to the Projects tab "Minecraft Server")

As I started feeling more familiar with the programming pipeline in Java, I went on to create some
smaller projects where I interfaced with new and unfamiliar APIs.
These projects were meant to help me better understand how the Java syntax works.
Below are a few tools i made during this time.
A rewrap of minecrafts server chat, allowing me to implement a live chat translator for people playing on my server.
A comunication bridge between discord and minecraft for server chats.
Some discord server tools like image grabbers and tools for easy lookups on statistics from databases I made.

After I had my java phase, I tried my hands on making games instead.
This phase started around August 2023, where I enrolled a game development school.
The course I took emphasized on touching all aspects of game development to learn how to make games independently.
Here I learned alot about the development pipeline for creating a game, and since I already had an invested interest in programming
I took extra time to learn how games are made from a programmers perspective and ended up learning alot, to the point where I released my own game on Steam.
